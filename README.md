
# Mango Monopoly

Mango Monopoly is a real estate-focused application powered by **Go** and **Python**. It provides users with an intuitive platform to view properties upcoming in local counties' real estate tax auctions and offers detailed information about those properties.

## Purpose

- **View Upcoming Tax Auctions**: Browse properties scheduled for tax auctions in your local counties.
- **Property Details**: Access comprehensive information about each property, including location, size, valuation, and more.

## Technologies Used

- **Go**: Backend development and server-side logic.
- **Python**: Data processing and property information scraping.

## Future Features

- **Expanded Real Estate Tools**: Integration of additional features such as property comparison and investment analysis.
- **User Accounts**: Ability to create profiles, save favorite properties, and receive notifications.
- **Interactive Maps**: Visual representation of properties on interactive maps for better geographical context.
